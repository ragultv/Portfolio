# My Portfolio

A modern, responsive portfolio website built with React and Vite, showcasing my projects, skills, and experience.

## 🚀 Features

- **Modern Design**: Clean and professional UI with particle animations
- **Responsive Layout**: Optimized for all device sizes
- **Interactive Components**: Smooth animations and hover effects
- **PDF Resume Viewer**: Integrated resume display and download functionality
- **Project Showcase**: Detailed project cards with images and descriptions
- **Skills Section**: Visual representation of technical skills
- **Contact Form**: Easy way for visitors to get in touch

## 🛠️ Built With

- **React 19** - Modern React with latest features
- **Vite** - Fast build tool and development server
- **React Bootstrap** - Responsive UI components
- **React Router** - Client-side routing
- **React PDF** - PDF document rendering
- **React Icons** - Icon library
- **TSParticles** - Interactive particle animations
- **Typewriter Effect** - Animated text effects

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## 🔧 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## 📁 Project Structure

```
src/
├── components/
│   ├── About/          # About section components
│   ├── Home/           # Home page components
│   ├── Projects/       # Projects showcase
│   ├── Resume/         # Resume viewer
│   ├── Footer.jsx
│   ├── Navbar.jsx
│   └── Particle.jsx
├── Assets/             # Images and static files
├── App.jsx            # Main application component
└── main.jsx           # Application entry point
```

## 🎨 Customization

1. **Personal Information**: Update content in respective component files
2. **Projects**: Add your projects in `src/components/Projects/Projects.jsx`
3. **Resume**: Replace `src/Assets/resume.pdf` with your resume
4. **Styling**: Modify CSS files for custom styling
5. **Colors**: Update theme colors in CSS variables
